# flux1-dev-fp8.safetensors 模型注释

**文件用途**：保存为同名 `.txt` 文件（`flux1-dev-fp8.txt`），放入 `models/checkpoints` 目录，可被 ComfyUI 模型管理插件自动读取，快速获取模型核心信息、使用参数与避坑指南。

## 一、 基础信息

- **模型名称**：FLUX.1 (dev) FP8
- **版本**：dev（开发版）· FP8 量化优化版
- **作者/团队**：Black Forest Labs（BFL）
- **基础架构**：FLUX.1 专属独立架构（非 SD1.5/SDXL，无兼容关系）
- **模型类型**：Full Checkpoint（FP8 轻量化量化完整模型）
- **文件大小**：约 8GB（相比原版 FLUX1-dev 瘦身 40%+，显存占用同步降低 40%）
- **核心特点**：高分辨率生成能力突出，量化优化后显存门槛降低、生成速度提升，兼顾超写实画质与中端设备流畅运行，是 FLUX 生态入门优选轻量化模型
- **开发定位**：超写实创作、高分辨率细节呈现、复杂场景构图，适合追求画质与效率平衡的中端显卡用户

## 二、 适用场景

| 风格类型                     | 适配程度  | 创作建议                                               |
| ---------------------------- | --------- | ------------------------------------------------------ |
| 超写实人像/风景/产品         | ✅ 极佳   | 核心优势场景，皮肤纹理、光影层次、材质表现接近真实照片 |
| 2048×2048 高分辨率创作       | ✅ 极佳   | 支持高分辨率直出，复杂场景逻辑一致性强，无明显变形     |
| 科幻/奇幻概念艺术/游戏 CG    | ✅ 优秀   | 氛围感拉满，细节丰富，适合打造史诗级场景与角色         |

## 三、 推荐参数配置（适配 ComfyUI）

| 参数类别               | 核心配置                                                     | 补充说明（FP8 版专属优化要点）                                                                  |
| ---------------------- | ------------------------------------------------------------ | ----------------------------------------------------------------------------------------------- |
| **分辨率**             | 1024×1024（基础）、1280×960、2048×2048（高分辨率）           | 8GB 显存流畅运行 1024×1024；12GB+ 显存支持 2048×2048 直出；6GB 显存需关闭多余程序尝试 1024×1024 |
| **采样器**             | 首选：FLUX Sampler（DDIM）<br>备选：FLUX Progressive Sampler | 必须使用 FLUX 专属采样器，不兼容 SD 系列（DPM++/Euler 等）采样器                                |
| **采样步数**           | 20-40 步（基础生成）、15-25 步（高分辨率修复）               | 30 步为黄金平衡点，FP8 版收敛速度更快，步数超过 40 无明显细节提升，且增加渲染时间               |
| **CFG Scale**          | 1.0-3.0（推荐 2.0）                                          | FLUX 模型对 CFG 值不敏感，无需拉高；＞ 3.0 易出现画面冗余、细节混乱                             |
| **Denoising Strength** | 0.3-0.5（高分辨率修复/放大）                                 | 低强度降噪优先，保留更多原生细节，避免画面失真、色彩偏移                                        |
| **显存优化**           | 启用 xformers、关闭多余自定义节点、降低批次大小              | FP8 版已做显存优化，搭配 xformers 可进一步降低 10%-15% 显存占用                                 |
| **推荐配套**           | FLUX 专属 ControlNet、FP8 优化 VAE                           | 无需额外安装第三方 VAE，模型内置色彩优化，按需搭配 FLUX 专属 ControlNet 提升构图准确性          |

## 五、 兼容性说明

- **基础框架**：✅ 仅兼容 FLUX.1 专属生态；❌ 不兼容 SD1.5/SDXL 及所有 SD 衍生框架
- **前端工具**：✅ 仅支持 ComfyUI（需安装 FLUX 专属自定义节点）；❌ 不支持 Automatic1111、InvokeAI
- **扩展插件**：✅ 支持 FLUX 专属 ControlNet（需对应 FLUX 版本预处理器与模型）；❌ 不兼容 SD 系列 ControlNet
- **LoRA/Embedding**：✅ 仅兼容 FLUX 专属 LoRA（SD 系列 LoRA 完全不兼容），LoRA 权重建议 0.5-0.7
- **硬件要求**：最低 6GB 显存（1024×1024 分辨率，关闭多余程序）；推荐 12GB+ 显存（支持 2048×2048 高分辨率、批量出图）
- **系统要求**：Windows 10/11（64 位）、Linux（Ubuntu 20.04+）；macOS 仅支持 M1/M2/M3 芯片（需额外配置）

## 六、 使用心得与注意事项

1.  **前置准备关键**：使用前必须在 ComfyUI 安装 FLUX 专属自定义节点（如 `FLUX Loader`、`FLUX Sampler`），否则无法加载模型或正常出图。
2.  **提示词技巧**：FLUX 模型支持自然语言描述，无需按照 SD 格式堆砌关键词，简洁清晰的语句（50 字以内最佳）即可获得优质效果，重点突出主体与风格。
3.  **显存控制优先级**：8GB 显存用户请勿直接尝试 2048×2048 分辨率，优先选择 1024×1024，如需放大可通过「低分辨率生成+小幅放大」实现，避免爆显存。
4.  **FP8 版与原版差异**：日常创作、中等复杂场景下画质与原版几乎无差异；极端复杂场景（多人物+精细特效+超大背景）下，细节表现略逊于原版，但效率优势显著。
5.  **避免常见踩坑**：① 勿使用 SD 系列采样器，会导致出图模糊或失败；② 勿拉高 CFG 值（＞ 3.0），会导致画面冗余、失去质感；③ 勿混用 SD 系列 LoRA/Embedding，无效果且可能报错。
6.  **保存格式建议**：高分辨率生成后，优先保存为 PNG 格式，保留更多细节与色彩信息，避免 JPG 压缩导致画质损失。
